Using the Pagila repo as a base 
https://github.com/devrimgunduz/pagila.git

SQL file with simple queries according the tasks given:
1. Display the number of films in each category, sorted in descending order.

2. Display the 10 actors whose films rented the most, sorted in descending order.

3. Display the category of films on which they spent the most money.

4. Print titles of movies that are not in inventory. Write a query without using the IN operator.

5. Display the top 3 actors who have most appeared in films in the “Children” category. If several actors have the same number of films, output all.

6. Display cities with the number of active and inactive customers (active — customer.active = 1). Sort by the number of inactive clients in descending order.

7. Print the category of movies that has the largest number of hours of total rental in cities (customer.address_id in this city), and that starts with the letter “a”. Do the same for cities in which there is a “-” symbol. Write everything in one request.
